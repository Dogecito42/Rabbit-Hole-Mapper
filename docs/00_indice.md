# Documentacion de desarrollo — RabbitHole Mapper

Esta carpeta contiene la guia de implementacion semana a semana y la referencia tecnica del proyecto.
Cada fichero puede leerse de forma independiente, pero el orden numerico sigue la secuencia logica de desarrollo.

## Ficheros

- 01_setup.md              Entorno, dependencias e instalacion de servicios externos
- 02_semana1_crawler.md    Capa 1: extraccion de datos con Playwright
- 03_semana2_agentes.md    Capa 2: agentes LLM con Ollama y Llama 3.1 8B
- 04_semana3_neo4j.md      Capa 3 (parcial): integracion con Neo4j y modelo de datos
- 05_semana4_qlearning.md  Capa 3 completa: Q-Learning y funcion de recompensa
- 06_semana5_simulacion.md Simulacion larga con los 3 agentes simultaneos
- 07_semana6_analisis.md   Capa 4: analisis Big Data (PageRank, Louvain, Shortest Path)
- 08_mvp.md                Checklist del minimo producto viable
- 09_resultados.md         Metricas y resultados esperados al final del proyecto
- 10_limitaciones.md       Limitaciones conocidas y lineas de trabajo futuro

## Regla de uso

Cada fichero de semana tiene una seccion "Entregable de la semana" al final.
Cuando ese entregable funciona, la semana se considera completada y se puede pasar a la siguiente.
