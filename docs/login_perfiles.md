# Login inicial de perfiles

Ejecutar en orden. Cada comando abre una ventana visible del navegador para autenticar con Google.
Completar el login manualmente y esperar a que el crawler termine los 5 videos.

## Perfil casual

```
python main.py --perfil casual --videos 5 --headless false
```

## Perfil gamer

```
python main.py --perfil gamer --videos 5 --headless false
```

## Perfil info

```
python main.py --perfil info --videos 5 --headless false
```
